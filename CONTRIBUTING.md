# Contributing to Diachronic Event Detection

Thank you for your interest in contributing! This project is part of an M2 research internship focusing on NLP and diachronic embeddings.

## How to Contribute

1.  **Report Bugs:** Use GitHub Issues to report bugs.
2.  **Suggest Enhancements:** Open an issue to discuss new features.
3.  **Pull Requests:**
    *   Fork the repository.
    *   Create a new branch (`git checkout -b feature/your-feature`).
    *   Commit your changes (`git commit -m 'Add some feature'`).
    *   Push to the branch (`git push origin feature/your-feature`).
    *   Open a Pull Request.

## Coding Style

*   Follow PEP 8 for Python code.
*   Document your functions and classes.
*   Keep the structure organized (src for scripts, notebooks for exploration).

## Research Context

Please keep in mind that this is a research project. Methodology changes should be discussed before implementation.

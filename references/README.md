# Research Bibliography (Core Papers)

### 1. Diachronic Word Embeddings and Semantic Shifts: A Survey (Kutuzov et al., 2018)
Provides a comprehensive overview of how word embeddings track language change over time.

### 2. CamemBERT: a Tasty French Language Model (Martin et al., 2020)
Introduces the CamemBERT model used in this project for French contextual embeddings.

### 3. Monitoring Semantic Shift with Diachronic Contextualized Word Embeddings (Martinc et al., 2020)
Explores using BERT-based models to detect changing word meanings.

### 4. Selective review of offline change point detection methods (Truong et al., 2020)
Review of the math behind PELT and Binary Segmentation used in our analysis.
